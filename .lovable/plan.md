

# The TVS School – Interactive Campus Guide

## Overview
A fully static, 3-page interactive website that helps parents and visitors navigate The TVS School campus with step-by-step directions, a smart search engine, and a warm, trustworthy design.

---

## Page 1 – Welcome / Hero Page

- **Full-width hero section** with the uploaded campus image as background, parallax scroll effect
- Animated overlay elements: floating map pin, compass, and footsteps icons
- Centered text: "Welcome to The TVS School" heading, welcoming subheading and description
- Two CTA buttons with subtle pulse animation: "Explore Our School" and "Find a Classroom"
- Smooth fade-in animations on load

## Page 2 – Mission & Vision

- Split-section layout with scroll-triggered animations (content fades in as you scroll)
- **Vision section** with prominent quote-style design
- **Mission section** as 6 icon cards with hover glow effects, each representing a mission pillar (sound academics, 21st-century skills, values, Indian heritage, teacher development, academic resources)
- Clean circular icons for each card

## Page 3 – Smart Campus Navigation (Core Feature)

- **Large search bar** with auto-suggest dropdown filtering all locations (classrooms, labs, offices, facilities)
- Hardcoded direction data for the entire campus (all the location data you provided)
- On selecting a location, display:
  - Starting point: "Main Entrance – Gate No 2"
  - Step-by-step direction cards with directional icons (arrow left/right/up, stairs icon)
  - Estimated walking time
  - Nearby landmarks
  - Highlighted animated path progression through the steps
- Card-based visual layout (no map image needed)
- "You are here" starting indicator

## Global UX Elements (All Pages)

- **Color scheme**: White background with light blue accents, school-themed warmth
- **Typography**: Large, readable, parent-friendly fonts
- **Breadcrumb navigation** at the top of every page
- **"Back to Entrance" button** always visible
- **Floating "Need Help?" button** with quick directions popup
- **Smooth animations**: fade-in, slide-up, hover highlights throughout
- **Mobile-first responsive design**
- **Accessibility**: large tap targets, good contrast

## Footer (All Pages)

- School logo area
- "Happy Schooling… This is My School" tagline
- Contact info placeholder
- Emergency help button
